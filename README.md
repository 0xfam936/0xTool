# 0xTool

0xTool is a cross-platform toolkit I developed — because why not? It’s simple to use and includes a variety of tools:

- Network scanner
- Portscanner
- MAC changer (Windows only)
- SSH client
- DOS
- MITM attack
- Webscraper
- Zip cracker
